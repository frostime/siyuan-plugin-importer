# 导入工具

![](https://raw.githubusercontent.com/terwer/siyuan-plugin-importer/main/public/icon.png)

导入 epub, docx, html 等导入到思源笔记。目前支持的格式：.epub, .docx, .html, .opml。

## 核心特色

- **选择笔记**：支持选择笔记本，下次打开可自动记忆上次选择的笔记本
- **一键导入**：选择文件，一键导入即可，零配置
